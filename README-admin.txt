# はじめにお読みください（管理者向け）

## url

http(s)://サーバドメイン/コンテキストパス/admin.phpになります。

## 設定変更方法

- 通知曜日
  ログイン後の画面から、実行制御のドロップダウンを変更し、
  「設定を更新する」をクリック --> 成功すると新しい内容が読み込まれる
- アラートしない
  ログイン後の画面の下部にある、yyyy年mm月dd日までアラートしないを
  入力し、チェックボックスにチェックした状態で「設定を更新する」を
  クリック --> 成功すると設定後の内容が読み込まれる

- メールの通知先、内容など（shell/mail.datを変更する）

- TITLE_TEXT --> TITLEに表示する項目を変更できます
- BODY_TEXT --> メール本文を設定できます
- FROM_ADDRESS --> メールの送信者アドレスを設定できます
- TO_ADDRESS --> メールの送信先アドレスを設定できます
